วิธีเขียน Prompt ที่ดี

หลักการสำคัญในการเขียน Prompt ที่ได้ผลลัพธ์ดีที่สุด

ระบุ Role & Context

บอก Copilot ว่าให้ทำหน้าที่อะไร เช่น "ในฐานะนักการตลาด..." หรือ "สมมติว่าคุณเป็นผู้เชี่ยวชาญด้าน..."

ตัวอย่าง ในฐานะนักเขียน copywriter มืออาชีพ ช่วยเขียนสโลแกนสินค้า...

กำหนด Format ผลลัพธ์

ระบุรูปแบบที่ต้องการ เช่น ตาราง, bullet points, หัวข้อ, ความยาว หรือจำนวนข้อ

ตัวอย่าง ...ตอบเป็นตาราง 3 คอลัมน์: หัวข้อ, รายละเอียด, ข้อดี ไม่เกิน 10 แถว

ให้ข้อมูล Context เพียงพอ

ยิ่งให้ข้อมูลพื้นหลังมาก ผลลัพธ์ยิ่งตรงเป้า บอกกลุ่มเป้าหมาย วัตถุประสงค์ ข้อจำกัด

ตัวอย่าง กลุ่มเป้าหมายคือผู้จัดการระดับกลาง อายุ 30-45 ปี ในอุตสาหกรรมการผลิต...

ใช้ Few-shot Examples

ให้ตัวอย่าง Input/Output ที่ต้องการ Copilot จะเรียนรู้รูปแบบและสร้างผลลัพธ์ที่คล้ายกัน

ตัวอย่าง ตัวอย่าง: Input: "สับปะรด" → Output: "สดชื่น หวาน เปรี้ยวนิดๆ"

Iterate & Refine

อย่าหยุดที่ครั้งแรก ใช้ผลลัพธ์แรกเป็นฐาน แล้ว refine ต่อ เช่น "ปรับให้เป็นทางการขึ้น" หรือ "เพิ่มข้อมูลส่วนที่ 2"

ตัวอย่าง ดี แต่ขอให้ส่วน Introduction สั้นลงครึ่งหนึ่ง และเพิ่มสถิติที่เกี่ยวข้อง

ใช้ Constraints ที่ชัดเจน

ระบุสิ่งที่ "ไม่ต้องการ" ด้วย เช่น ห้ามใช้คำเฉพาะ ความยาวสูงสุด หรือหัวข้อที่ไม่ต้องครอบคลุม

ตัวอย่าง ห้ามใช้ศัพท์เทคนิค เขียนไม่เกิน 200 คำ ไม่ต้องมีส่วน conclusion

มี Prompt ดีๆ อยากแบ่งปัน?

ร่วมเป็นส่วนหนึ่งของชุมชน ส่ง Prompt ของคุณมาเพื่อช่วยเหลือนักพัฒนาและผู้ใช้คนอื่น
