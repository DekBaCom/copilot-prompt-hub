Copilot Cowork คืออะไร?

Copilot Cowork คือระบบ Agentic AI ที่ execute งานซับซ้อน ระยะยาว และใช้หลาย tool พร้อมกันแบบ end-to-end — ไม่ใช่แค่ draft หรือ recommendation แต่ส่งผลลัพธ์จริง ทำงานบน cloud ต่อเนื่องแม้ปิดหน้าจอ

☁️ Cloud-hosted — ทำงานต่อเนื่องตลอดเวลา
🔐 Enterprise Security บน M365 Trust Boundary
🧠 Multi-model (Opus 4.8, Sonnet 4.6, Cowork 1)
💰 30–40% ถูกกว่า Claude Cowork + M365 connector

⚡ Task Types & Credit Estimate

ต้นทุน Credits ขึ้นอยู่กับ Model, Context, Tool Calls และ Runtime

⚡ Light ต้นทุนต่ำ
  • แหล่งข้อมูลน้อย / Knowledge source เดียว

  • ≥ 2 outputs

  • Broad aggregation ข้ามระบบ
  • Deep reasoning หลายขั้นตอน
  • หลาย outputs + Tool calls ซับซ้อน
เช่น Financial report, Multi-system research

วิธีเขียน Prompt ที่ดี

หลักการสำคัญในการเขียน Prompt ที่ได้ผลลัพธ์ดีที่สุด

ระบุ Role & Context

บอก Copilot ว่าให้ทำหน้าที่อะไร เช่น "ในฐานะนักการตลาด..." หรือ "สมมติว่าคุณเป็นผู้เชี่ยวชาญด้าน..."

ตัวอย่าง ในฐานะนักเขียน copywriter มืออาชีพ ช่วยเขียนสโลแกนสินค้า...

กำหนด Format ผลลัพธ์

ระบุรูปแบบที่ต้องการ เช่น ตาราง, bullet points, หัวข้อ, ความยาว หรือจำนวนข้อ

ตัวอย่าง ...ตอบเป็นตาราง 3 คอลัมน์: หัวข้อ, รายละเอียด, ข้อดี ไม่เกิน 10 แถว

ให้ข้อมูล Context เพียงพอ

ยิ่งให้ข้อมูลพื้นหลังมาก ผลลัพธ์ยิ่งตรงเป้า บอกกลุ่มเป้าหมาย วัตถุประสงค์ ข้อจำกัด

ตัวอย่าง กลุ่มเป้าหมายคือผู้จัดการระดับกลาง อายุ 30-45 ปี ในอุตสาหกรรมการผลิต...

ใช้ Few-shot Examples

ให้ตัวอย่าง Input/Output ที่ต้องการ Copilot จะเรียนรู้รูปแบบและสร้างผลลัพธ์ที่คล้ายกัน

ตัวอย่าง ตัวอย่าง: Input: "สับปะรด" → Output: "สดชื่น หวาน เปรี้ยวนิดๆ"

Iterate & Refine

อย่าหยุดที่ครั้งแรก ใช้ผลลัพธ์แรกเป็นฐาน แล้ว refine ต่อ เช่น "ปรับให้เป็นทางการขึ้น" หรือ "เพิ่มข้อมูลส่วนที่ 2"

ตัวอย่าง ดี แต่ขอให้ส่วน Introduction สั้นลงครึ่งหนึ่ง และเพิ่มสถิติที่เกี่ยวข้อง

ใช้ Constraints ที่ชัดเจน

ระบุสิ่งที่ "ไม่ต้องการ" ด้วย เช่น ห้ามใช้คำเฉพาะ ความยาวสูงสุด หรือหัวข้อที่ไม่ต้องครอบคลุม

ตัวอย่าง ห้ามใช้ศัพท์เทคนิค เขียนไม่เกิน 200 คำ ไม่ต้องมีส่วน conclusion

Copilot License vs Copilot Credits คืออะไร?

ทำความเข้าใจความแตกต่างของ Licensing ก่อนเลือกแผน

🪪

Copilot License

Per-User · จ่ายรายเดือนตามจำนวน User

License รายบุคคลที่ assign ให้กับผู้ใช้แต่ละคน เพื่อเปิดใช้งาน Copilot AI ใน Microsoft 365 apps และบริการองค์กร จ่ายตามจำนวน user ที่ต้องการใช้งาน ไม่ขึ้นกับปริมาณการใช้งานจริง

$18 – $30 /user/เดือน

Services ที่ต้องใช้ Copilot License

  • 📝 Copilot ใน Word, Excel, PowerPoint
  • 📧 Copilot ใน Outlook (Email Draft & Summary)
  • 💬 Copilot ใน Teams (Meeting Summary & Recap)
  • 📓 Copilot ใน OneNote & Loop
  • 🌐 Microsoft 365 Chat / BizChat (Work context)
  • 📄 Copilot Pages
  • 💼 Copilot ใน Viva (Insights, Learning)
  • 🔍 SharePoint Copilot & Copilot Search
🪙

Copilot Credits

Consumption-based · จ่ายตามปริมาณการใช้จริง

หน่วยการใช้งานแบบ pay-per-use สำหรับ Copilot Cowork และ Copilot Studio นับตาม Model, Context, Tool Calls และ Runtime ของแต่ละ task เลือกจ่ายแบบ Pay-as-you-go หรือ Prepaid (P3) เพื่อรับส่วนลด

$0.01 / credit (Pay-as-you-go)

🤖 Copilot Studio — Agent Channels

  • 🌐 Agent บน Website / Web Chat (external users)
  • 💬 Agent บน Microsoft Teams (external / guest users)
  • 📱 Agent บน Mobile App ผ่าน Direct Line API
  • 📘 Agent บน Facebook Messenger, Slack, LINE
  • 📞 IVR / Voice Bot (Azure Communication Services)
  • 📧 Agent ผ่าน Email Channel

⚡ Automation & Advanced AI

  • 🔄 Autonomous Actions — AI ทำงานแทน user โดยอัตโนมัติ
  • 🔗 Multi-agent Orchestration (Agent เรียก Agent อื่น)
  • 🧠 Custom Engine Agent (Azure OpenAI + Copilot Studio)
  • ⚡ Power Automate Premium flows ที่ trigger โดย Agent

➕ กรณีเกินโควตาที่รวมใน M365 Copilot

  • 🏢 Internal Agent sessions เกินโควตา tenant รายเดือน
  • 📊 Dataverse queries และ Connector calls ปริมาณสูง
💡
เมื่อไรที่ต้องใช้ทั้งสองอย่าง?

องค์กรที่ต้องการทั้ง Copilot ใน M365 Apps และ สร้าง Agent สำหรับลูกค้าภายนอก จำเป็นต้องมีทั้ง Copilot License (สำหรับ user ภายใน) และ Copilot Credits (สำหรับ session ของ Agent ที่ให้บริการลูกค้า)
